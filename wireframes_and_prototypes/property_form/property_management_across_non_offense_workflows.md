Property Management Across Non‑Offense Workflows

Purpose

This document explains why Property should be available and consistent across all non‑offense workflows in OTIS 2.0 and provides guidelines for its use.

Rationale

- Consistent Experience: Troopers and staff should not have to guess which forms include the property tab. Many non‑offense cases (e.g. injured person, mental disability, assist other agency) can involve property that must be seized, stored, or returned.
- Chain‑of‑Custody Integrity: Property handling requires documentation regardless of the nature code. Having property features in every non‑offense case ensures that chain of custody remains intact.
- Simplified Training: When the property tab is always available, training can focus on how to use it rather than when it appears.

Guidelines

1. Always Accessible: Enable the property tab for all non‑offense case types. It should appear by default in the case menu, not conditionally based on the nature code.
2. Optional vs. Required: Property entry should be optional unless the nature code specifically requires it (e.g. recovered property, vehicle seizure). The user should be able to save a case with no property items recorded.
3. Consistency in Fields: Use consistent fields across case types: item description, quantity, value, condition, serial number, make/model, location of recovery, and current storage location. Provide checkboxes or drop‑downs for special categories (e.g. firearms, currency, vehicles).
4. Linking to Person & Vehicle: Allow property items to be linked to persons (owner, finder) and vehicles as needed. For example, property found on a person in an injured person case should be linked to that person record.
5. Disposition Tracking: Provide disposition options that reflect non‑offense cases (e.g. returned to owner, destroyed, retained, transferred to another agency). The same disposition codes should apply whether the property was involved in an offense or not.
6. LEADS Integration: For property requiring LEADS entry (e.g. lost badge, lost firearm, seized currency), prompt the user to provide the LEADS entry reference.

Benefits

- Efficiency: Personnel do not need to create a separate recovered property case just to record property for a non‑offense incident.
- Accuracy: Ensures property is tracked consistently, improving accountability and reducing misplaced or unreturned items.
- Compliance: Maintains chain‑of‑custody records in compliance with evidence and public records requirements.
